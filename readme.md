# cbrt

> ES2015 [`Math.cbrt()`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Math/cbrt) [ponyfill](https://ponyfill.com)


## Install

```
$ npm install cbrt
```


## Usage

```js
const cbrt = require('cbrt');

cbrt(1000);
//=> 10
```
